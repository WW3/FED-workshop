KB + SR Accessible carousel

use the left and right key to move forward and backwords through the slides

Accessibility features

  • Carousel keybaord operability
  • Buttons keyboard operability
  • Buttons semantics
  • Buttons visual focus
  • Focus management
  • Hide the invisble carousel items from keyboard users (tabindex="-1")
  • Enable items navigation with the keyboard left & right arrow keys
  • Carousel accessible name (aria-label) + description (aria-describedby)
  • Buttons accessible name (aria-label)
  • Hide invisble slides from screen reader users (aria-hidden="true")
  • Announce new visible slide (text), on previous/next actions (aria-live)
